YellowBall 개인정보 처리방침

시행일: 2026년 5월 4일

YellowBall은 테니스 및 피클볼 스트링 작업 예약, 라켓 관리, 데모 라켓 예약, 알림 제공을 위해 필요한 최소한의 개인정보를 처리합니다.

1. 수집하는 개인정보

  • 계정 정보: 이메일, 닉네임, 전화번호, 사용자 ID
  • 예약 정보: 예약 서비스, 예약 시간, 요청사항, 예약 상태 기록
  • 라켓 정보: 라켓 브랜드, 모델, 사진, 스트링 세팅 정보
  • 배송 또는 방문 관리 정보: 사용자가 입력한 주소
  • 알림 정보: 푸시 알림 토큰, 알림 수신 설정, 알림 읽음 상태
  • 운영 및 보안 정보: 관리자 처리 기록, 계정 상태, 부정 이용 방지를 위한 예약 및 노쇼 기록

2. 개인정보 이용 목적

  • 회원 가입, 로그인, 계정 관리
  • 스트링 작업 및 데모 라켓 예약 생성, 조회, 변경, 취소
  • 예약 상태, 운영 공지, 관리자 알림 발송
  • 고객 문의 대응 및 서비스 품질 개선
  • 부정 이용 방지, 보안 감사, 법령상 의무 이행

3. 개인정보 보관 기간

개인정보는 서비스 제공 목적 달성 시까지 보관하며, 계정 삭제 요청 시 다음과 같이 처리합니다.

  • 삭제 요청 후 30일간 대기 기간이 적용됩니다. 이 기간 동안 서비스 접근은 제한되며, 사용자가 요청을 철회하지 않으면 30일 경과 후 개인정보(닉네임, 전화번호, 주소, 라켓, 알림 설정 등)가 자동으로 영구 삭제 또는 비식별 처리됩니다.
  • 예약, 결제, 고객 대응, 감사 기록처럼 법령상 보관이 필요한 정보는 해당 법령이 정한 기간 동안 보관할 수 있습니다.
  • 삭제가 완료된 데이터는 복구할 수 없습니다.

4. 제3자 제공 및 처리 위탁

YellowBall은 사용자의 개인정보를 임의로 판매하지 않습니다. 서비스 운영을 위해 다음 외부 서비스를 사용할 수 있습니다.

  • Supabase: 인증, 데이터베이스, 스토리지, Edge Functions
  • Expo: 모바일 앱 빌드, 푸시 알림 전송 인프라
  • Google Play: Android 앱 배포 및 결제, 보안, 통계 제공

외부 서비스는 각 서비스의 개인정보 처리방침과 보안 기준에 따라 정보를 처리합니다.

5. 사용자의 권리

사용자는 앱에서 프로필 수정, 알림 설정 변경, 계정 삭제 요청을 할 수 있습니다. 개인정보 열람, 정정, 삭제, 처리 정지를 요청하려면 앱 내 계정 삭제 화면 또는 아래 연락처로 문의할 수 있습니다.

6. 아동 개인정보

YellowBall은 법정대리인의 동의 없이 아동의 개인정보를 고의로 수집하지 않습니다. 관련 정보가 확인되면 지체 없이 삭제합니다.

7. 보안 조치

YellowBall은 인증, Row Level Security, 접근 권한 분리, 감사 로그, 안전한 토큰 저장 방식을 통해 개인정보 접근을 제한합니다.
